# WhatsApp外贸号码查询系统

专为工程机械外贸团队设计的WhatsApp号码智能查询工具。

## 🌟 功能特点

- 📱 **智能号码格式化** - 自动识别并格式化各国WhatsApp号码
- 🌍 **200+国家数据库** - 覆盖全球主要外贸市场
- ⏰ **实时时区查询** - 显示客户当地时间和工作时间状态
- 🗣️ **多语言信息** - 显示各国主要语言
- 💱 **货币信息** - 显示各国货币代码和名称
- 📋 **一键复制** - 快速复制格式化号码
- 💬 **WhatsApp直达** - 一键打开WhatsApp聊天
- 📜 **查询历史** - 本地保存查询记录

## 🚀 在线使用

访问：https://你的域名.vercel.app

## 🛠️ 技术栈

- React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Vite

## 📦 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📝 使用说明

1. 在输入框中输入客户WhatsApp号码（支持多种格式）
2. 点击「查询」按钮
3. 查看格式化结果、国家信息、当地时间等

**支持的输入格式：**
- `+8613812345678`
- `8613812345678`
- `008613812345678`
- `138-1234-5678`
- `138 1234 5678`
